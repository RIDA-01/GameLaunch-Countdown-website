A responsive, animated landing page for the upcoming battle royale mobile game "Last Man Standing" by Rida Games. Built with modern web technologies and featuring a countdown timer, email subscription forms, and smooth animations.

## Features

- **Countdown Timer** - Real-time countdown to the game launch date (February 25, 2026)
- **Smooth Animations** - Powered by WOW.js and Animate.css for engaging scroll animations
- **Modern UI** - Clean, game-themed design with custom icons and typography
- **Social Media Integration** - Links to Facebook, Twitter, LinkedIn, and Instagram
- **Bootstrap Framework** - Built on Bootstrap for robust grid system and components

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Custom styling with animations
- **JavaScript (jQuery)** - Interactive functionality
- **Bootstrap 4** - Responsive grid and components
- **WOW.js** - Scroll animations
- **Animate.css** - CSS animations
- **jQuery Countdown** - Launch timer functionality
- **Font Awesome** - Social media icons
